
FACEFIT - JOY365 | אפליקציית התאמה חכמה

📁 מבנה התיקייה:
- index.html — קובץ ראשי של האפליקציה
- manifest.json — הגדרות PWA כולל שם, צבעים ואייקונים
- service-worker.js — קובץ המטפל בזכרון מטמון ודפדפן
- icon-192.png / icon-512.png — אייקונים מותאמים למסך הבית

📤 שלבים להעלאה ב-GitHub:
1. התחברי ל-GitHub וצרי ריפוזיטורי חדש בשם facefit
2. העלי את כל הקבצים האלו ישירות לריפו דרך כפתור "Upload files"
3. בצעי Commit

🚀 העלאה ל-Vercel:
1. התחברי ל-vercel.com עם חשבון GitHub
2. לחצי Add New → Project → בחרי את הריפו facefit
3. המתיני לסיום הבנייה (המערכת תזהה index.html לבד)
4. קבלי לינק דינמי → גשי אליו מהטלפון ולחצי "הוסף למסך הבית"

💡 טיפ:
אם את רוצה להחליף את האייקון, פשוט החליפי את הקובץ icon-512.png ואל תשכחי לעדכן גם ב-manifest.json

✨ בהצלחה!
